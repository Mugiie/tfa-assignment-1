HTML Attributes:

HTML attributes are special words used inside the opening tag to control the element's behaviour.
HTML attributes are a modifier of an HTML element type.

  • Alt attributes can be beneficial for the search engine and browser.

    <img src=“Source” alt=“ This is an image of a computer ”/>
    if the browser was unable to load the image it will show the alt text.


  • The src attribute specifies the path to the image to be displayed.

    <img src=“Source” alt=“ This is an image of a computer ”/>


  • Width and height attributes, which specifies the width and height of the image/video.

    <img src=“Source” alt=“ This is an image of a computer ” width=“500” height=“500” />


  • The title attribute defines some extra information about an element.

    <p title=“Hello ”>This is a paragraph.</p>
  • Action - Specifies where to send the form-data when a form is submitted.


  • controls - Specifies that audio/video controls should be displayed (such as a play/pause button etc).


  • For- Specifies which form element(s) a label/calculation is bound to.


  • Href- Specifies the URL of the page the link goes to.
